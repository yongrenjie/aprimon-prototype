Welcome!

This website will (try to) determine what Aprimon you can request from a trading partner, i.e. what they have but you don't have.

  • If you want the opposite (i.e. what you can offer them) please use the swap button and submit again

  • BDSP doesn't work for now, I need to manually enter legality for each Pokemon.....

  • Please note that this only checks a spreadsheet for the presence of a ticked box — it doesn't do anything fancy such as looking for HA availability or pending trades. It is still your responsibility to check for this (if it matters to you, e.g. for genderless/monogender species), or even better, inform your trade partner if you don't have HA on something!

  • The comparison also combines entries from regional forms (e.g. Kanto and Alola Vulpix) and other things which can be crossbred (e.g. Basculin colours or Nidoran gender)

  • This is rather preliminary — if there are any issues please let me know — my Reddit username is /u/orthocresol.

  • You can find the source code on GitHub too (please feel free to submit issues or pull requests there if you are thus inclined!).
